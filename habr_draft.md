# Как я сделала сайт-конвертер за 250 рублей с помощью Claude Code и планирую обогнать конкурентов в Яндексе

Привет, Хабр! Меня зовут Алия, и я хочу рассказать историю о том, как мечта из детства, сломанные QR-сканеры в рунете и AI-ассистент в терминале привели к запуску сайта с семью онлайн-инструментами. Бюджет проекта — 250 рублей за домен. Всё остальное — бесплатно.

## Проблема: почему QR-сканеры в рунете не работают?

Всё началось банально. Мне нужно было расшифровать QR-код с картинки. Казалось бы — 2026 год, задача на 5 секунд. Открываю Яндекс, ввожу «расшифровать QR-код онлайн».

Первый сайт — просит установить приложение. Второй — загружает фото на сервер и выдаёт ошибку. Третий — вообще не работает, кнопка «Сканировать» ведёт в никуда. Четвёртый — показывает рекламу на весь экран, а под ней пустая страница.

Из любопытства попробовала то же самое в англоязычном Google — первый же результат отработал идеально. Загрузила картинку, получила ссылку. Три секунды.

**Почему так?** Мне кажется, многие русскоязычные утилиты-сайты были сделаны давно и заброшены. Они висят в топе поиска по инерции, но реально не работают. Или работают только частично — например, QR-сканер без поддержки камеры и буфера обмена.

## Мечта из детства

Сколько себя помню, хотела сделать сайт-утилиту — типа конвертера файлов, переводчика кодов, что-то полезное. И чтобы он зарабатывал. Не миллионы — хотя бы 3 000 рублей в месяц с рекламы. Просто приятно, когда твоя штука работает и приносит хоть какой-то доход.

И вот я нашла нишу: онлайн-инструменты для расшифровки в русском сегменте. QR-коды, двоичный код, Base64, HEX — конкуренты либо не работают, либо делают всё через сервер (привет, приватность), либо выглядят как сайт из 2008 года.

## Реализация: Claude Code + GitHub Pages

Весь сайт я написала с помощью **Claude Code** — это AI-ассистент, который работает прямо в терминале. Ты описываешь что хочешь, а он пишет код, создаёт файлы, коммитит в git. Можно общаться на русском.

Процесс выглядел примерно так:

— *«Сделай QR-сканер, который работает с картинками, камерой и буфером обмена»*
— Claude Code создаёт HTML-файл с полным функционалом
— *«Добавь двоичный код переводчик»*
— Готово, с UTF-8, кириллицей, двумя режимами
— *«Теперь Base64, HEX и URL — запусти три агента параллельно»*
— Три страницы создаются одновременно в фоне

За один вечер — 7 полноценных инструментов.

### Стек

Максимально простой:

- **Чистый HTML + CSS + JavaScript** — никаких фреймворков, бандлеров, node_modules
- **GitHub Pages** — бесплатный хостинг, деплой по push в main
- **Домен** — 250 рублей на reg.ru
- **jsQR** — единственная внешняя библиотека, для декодирования QR-кодов
- **Всё остальное** (MD5, Base64, Binary, HEX, URL) — чистый JS, работает в браузере

Никакого бэкенда. Данные пользователя никуда не отправляются. Можно отключить интернет — и всё продолжит работать (кроме загрузки шрифтов).

### Что умеет сайт

| Инструмент | Что делает |
|---|---|
| **QR Сканер** | Расшифровка QR-кода из картинки, буфера обмена или камеры |
| **Двоичный код** | Перевод между текстом и двоичным кодом (UTF-8) |
| **MD5** | Генерация MD5-хеша и проверка формата |
| **ASCII** | Интерактивная таблица всех 128 символов с поиском |
| **Base64** | Кодирование и декодирование Base64 |
| **HEX** | Конвертер шестнадцатеричного кода в текст |
| **URL** | URL encode/decode |

Все инструменты работают в реальном времени — вводишь текст, результат появляется мгновенно. Никаких кнопок «Конвертировать» и ожидания.

## SEO: как я планирую обогнать конкурентов

Я не просто сделала сайт и забыла. Я проанализировала конкурентов через Similarweb.

Главный конкурент — **decodeit.ru**. У них ~28 000 кликов в месяц из поиска. Топовые запросы:

- «двоичный код в текст» — 4 200 кликов/мес
- «расшифровка двоичного кода» — 2 600 кликов/мес
- «расшифровать двоичный код» — 1 900 кликов/мес
- «бинарный код в текст» — 1 000 кликов/мес

Все эти запросы — информационные, с высоким CTR на первую позицию. И конкуренция не такая высокая, как может показаться.

**Мой план:**

1. **Правильные мета-теги** — title, description, keywords заточены под реальные поисковые запросы из данных конкурентов
2. **Schema.org разметка** — WebApplication и FAQPage на каждой странице
3. **Больше страниц** — каждая утилита привлекает свой пул запросов. 7 страниц > 4 страницы
4. **Поведенческие факторы** — сайт быстрый, удобный, работает без загрузок на сервер. Пользователь получает результат мгновенно и не возвращается в поиск
5. **Яндекс.Вебмастер + Google Search Console** — отправила sitemap, жду индексацию
